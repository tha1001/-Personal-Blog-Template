# 📝 Personal Blog Template

A clean, responsive, and modern multi-page personal blog template built with **HTML5** and **CSS3**. This project focuses on creating a simple blogging interface with a modern UI, responsive layout, and reusable components.

---

## ✨ Features

- Responsive Design
- Modern User Interface
- Multi-page Layout
- Sticky Navigation Bar
- Blog Post Cards
- Single Post Page
- Categories Page
- About Page
- Contact Form UI
- Custom Persian Font (Vazirmatn)
- Smooth Animations & Hover Effects

---
